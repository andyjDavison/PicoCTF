# vault-door-4

## Description
This vault uses ASCII encoding for the password.

## Challenge Information
**Point Value:** 250  
**Category:** Reverse Engineering

## Hints
1. Use a search engine to find an "ASCII table".
2. You will also need to know the difference between octal, decimal, and hexadecimal numbers.

## Solution

We can change this method checkPassword() to print out this byte array for us like so:
```Java
public boolean checkPassword(String password) {
        byte[] passBytes = password.getBytes();
        byte[] myBytes = {
```

This gives us a string of decimals ```106 85 53 116 95 52 95 98 85 110 67 104 95 48 102 95 98 89 116 51 115 95 102 52 97 56 99 100 56 102 55 101``` that we can put into a decimal to ASCII converter and get our flag.
